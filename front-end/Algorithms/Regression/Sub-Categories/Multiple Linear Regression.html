About Multiple Linear Regression

Multiple linear regression (MLR), also known simply as multiple regression, is a statistical technique that uses several explanatory variables to predict the outcome of a response variable.

Requirements

If the dependent variable (y) consists of numeric values and there is more than one independent variable (X). Here, we have to use multiple Linear Regression.

Applications

  • Forecasting
    • Linear regressions can be used in business to evaluate trends and make estimates or forecasts.
  • Analysing
    • Linear regression can also be used to analyze the marketing effectiveness, pricing and promotions on sales of a product.
